kubectl create secret generic my-secret

Comment: Initialize a "Generic" secret type named my-secret. This is the standard choice for most app credentials.



--from-literal=username=admin

Comment: Create a key called username. By using --from-literal, we avoid needing an external file, which is great for quick setups.



--from-literal=password=123456

Comment: Create a key called password. Note that Kubernetes will automatically Base64 encode this value before storing it.
